# First Trial
주어진 순열의 다음 순열을 구하는 문제.

먼저 주어진 순열(input)과 다음 순열(output) 두 list를 생각해 보면, 각각의 순열을 두 부분으로 나눌 수 있다.

하나는 input, output에서 변하지 않는 부분이고,
두 번째는 변하는 부분, 즉 첫 번째 부분의 나머지 부분이다.

예를 들어, input이 `[1, 3, 5, 4, 2]` 라면, output은 `[1, 4, 2, 3, 5]`가 될 것이고, 변하지 않는 부분은 `[1]`, 변하는 부분은 `[3,5,4,2]`가 된다.

변하는 부분은 맨 뒤에서부터 오름차순이 아닌 index를 찾고 해당 index부터 slice 한 부분이 되고, (`[3,5,4,2]`)

가장 첫 번째 원소보다 다음으로 큰 값을 맨 앞에, 나머지를 오름차순으로 배열 (`[4,2,3,5]`)

이 결과를 변하지 않는 부분과 합치면 바로 다음 순열이 된다. (`[1,4,2,3,5]`)

다음 순열은 순열을 숫자로 생각할 때, 다음에 만들 수 있는 가장 작은 숫자이고, 곧 특정 위치에서 특정 숫자들을 작은 숫자대로 배열하는 것이라 생각하면 이해가 편함
